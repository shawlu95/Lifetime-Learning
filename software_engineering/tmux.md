Basics

```bash
tmux new -session_name
tmux new-session -session_name
exit
tmux attach -t session_name
<prefix> + d
tmux kill-session -t2
tmux ls
tmux kill-server
```

Window

```bash
# create window
<prefix> + C

# rename window
<prefix> + ,

# kill window
<prefix> + &

# vertical split
<prefix> + "

# horizontal split
<prefix> + %

# kill plane
<prefix> + x
```
